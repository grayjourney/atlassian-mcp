Create an API token, then paste your details below. The same token works for both Jira and Confluence on Atlassian Cloud.

  1. Go to https://id.atlassian.com/manage-profile/security/api-tokens and click Create API token.
  2. Copy the token and paste it into the matching field(s) below.
  3. Your username is the email you sign in to Atlassian with.
